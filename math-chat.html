생성형 AI를 활용한 중등 수학 개념 및 풀이 챗봇

RAG 기술을 활용하여 중학교 1학년 수학 개념과 문제 풀이를 단계별로 제공하는 AI 챗봇입니다.

GitHub 보기 2024.11.7 ~ 11.14

프로젝트 개요

KDT 개발 멘토로 참여한 결과물로, PDF 형태의 수학 교재를 벡터 데이터베이스에 저장하고, 학생의 질문에 맞는 관련 내용을 검색하여 GPT-4를 통해 친근하고 이해하기 쉬운 설명을 제공합니다.

기획팀: PM 7기 김민정, 김지민, 김화영, 유영채, 윤수정
개발 멘토: 김현경, 이신영, 이효준
역할: 개발 멘토

주요 특징

  • RAG 기반 PDF 교재 문서 검색
  • 단계별 수학 풀이 제공
  • 중학교 1학년 수준 맞춤 설명
  • 웹 기반 실시간 채팅 인터페이스

기획 및 개발 발표자료

프로젝트 개요

RAG 기반 수학 학습 챗봇 기획

시스템 구조

RAG 아키텍처 및 기술 스택

개발 결과

구현된 기능 및 성과

기술적 구현

RAG 3단계 전략 구현

기술 스택

Backend

  • • Flask (Python)
  • • OpenAI GPT-4o
  • • LangChain
  • • FAISS (벡터 데이터베이스)
  • • OpenAI Embeddings
  • • PyPDF2, PyPDF (문서 처리)

Frontend & Tools

  • • Flask Jinja2 (템플릿 엔진)
  • • CSS (스타일링)
  • • JavaScript
  • • Python 3.10.0
  • • VSCode, venv

주요 기능

RAG 기반 문서 검색

  • • PDF 교재를 청킹하여 벡터 DB에 저장
  • • MMR 방식으로 관련 문서 내용 검색
  • • 검색된 문서 기반 정확한 답변 생성

단계별 수학 풀이

  • • RAG 3단계 전략 대응 GPT 단계별 풀이
  • • 중학교 1학년 수준 친근한 말투
  • • 복잡한 개념을 단계별로 나누어 설명

적응형 학습 지원

  • • 추가 설명 요청 시 재설명 제공
  • • 학생 수준에 맞는 용어 사용
  • • 존댓말과 친근한 "~에요" 말투

웹 기반 채팅 인터페이스

  • • 실시간 채팅 형태의 질의응답
  • • 직관적이고 사용하기 쉬운 인터페이스
  • • 대화 기록 유지 및 연속적 학습 지원

데이터 처리 파이프라인

  • • PyPDFDirectoryLoader를 통한 PDF 문서 일괄 로딩
  • • RecursiveCharacterTextSplitter로 청크 분할
  • • OpenAI Embeddings를 통한 텍스트 벡터화
  • • FAISS 벡터 데이터베이스에 코사인 유사도 기반 저장

AI 알고리즘

  • • MMR 방식으로 관련 문서 검색
  • • Map-Reduce 체인을 통한 답변 생성
  • • 중학교 1학년 수준 프롬프트 엔지니어링
  • • 검색된 문서들을 결합한 일관성 있는 답변

역할 및 기여

개발 멘토 역할

  • • 시스템 설계: RAG 아키텍처 설계 및 구현 가이드
  • • 기술 지도: LangChain, FAISS, OpenAI API 활용 멘토링
  • • 코드 리뷰: 효율적인 벡터 검색 및 답변 생성 로직 개선
  • • 성능 최적화: 청킹 전략 및 검색 파라미터 튜닝

주요 기여 영역

  • • RAG 3단계 전략 구현 및 최적화
  • • 교육용 프롬프트 엔지니어링
  • • 벡터 데이터베이스 구축 및 관리
  • • 웹 인터페이스 개발 지원
