코스매쓰 - AI 수학 교육 플랫폼

KSM과 IRT 이론을 기반으로 한 초등수학 AI 코스웨어. 적응형 평가와 시각적 지식맵을 통한 개인화된 학습 경험을 제공합니다.

GitHub 보기 2024.11.27 ~ 2025.01.02

프로젝트 개요

KDT PM7기 교육생들이 기획한 초등수학 AI코스웨어 "코스매쓰" MVP 모델 제작 결과물입니다. 초등학교 수학 교육과정 중 '다각형' 단원을 대상으로 한 AI 기반 적응형 학습 시스템입니다.

기획팀: PM 7기 유수진, 박유진, 최은지, 정진용
개발팀: 이신영, 이효준
역할: 개발

주요 특징

  • 적응형 평가: 학습자의 답변 패턴 분석
  • 시각적 피드백: 지식맵을 통한 학습 진도 시각화
  • 다양한 문제 유형: 객관식, 주관식, 드래그앤드롭
  • 개인화된 학습 경험: 개별 수준 맞춤 콘텐츠

주요 기능

다양한 문제 유형 지원

  • • 객관식 문제: 선택지 중 정답 선택
  • • 주관식 문제: 직접 답안 입력
  • • 드롭다운 문제: 드롭다운에서 답안 선택
  • • 드래그앤드롭 문제: 요소를 드래그하여 정답 배치

실시간 학습 진도 추적

  • • 문제별 답변 상태 실시간 업데이트
  • • 진도율 시각화 (프로그레스 바)
  • • 이전/다음 문제 네비게이션

지식맵 시각화

  • • 개념 간 연관관계를 노드와 엣지로 표현
  • • 학습자의 이해도에 따른 색상 구분
  • • 인터랙티브한 지식 구조 탐색

개념별 성취도 분석

  • • 18개 문제를 통한 6개 핵심 개념 평가
  • • 다각형 기본 개념, 정다각형, 대각선
  • • 모양조각과 모양 만들기, 모양 채우기
  • • 개념별 정답률 및 취약점 분석

시스템 아키텍처

src/
├── components/           # React 컴포넌트
│   ├── questions/       # 문제 유형별 컴포넌트
│   │   ├── MultipleChoice.tsx
│   │   ├── ShortAnswerQuestion.tsx
│   │   ├── DropdownQuestion.tsx
│   │   └── DragDropQuestion.tsx
│   ├── EvaluationPage.tsx    # 평가 메인 페이지
│   ├── ResultPage.tsx        # 결과 페이지
│   ├── KnowledgeMap.tsx      # 지식맵 컴포넌트
│   └── CircleItem.tsx        # 지식맵 노드 컴포넌트
├── context/             # React Context
│   └── EvaluationContext.tsx # 평가 상태 관리
├── data/               # 정적 데이터
│   └── questions.ts    # 문제 데이터
├── types/              # TypeScript 타입 정의
└── App.tsx            # 메인 애플리케이션

데이터 흐름

  • • 문제 데이터 로드: questions.ts에서 18개 문제 데이터
  • • 상태 관리: EvaluationContext를 통한 답변 상태 관리
  • • 답변 처리: 각 문제 유형별 컴포넌트에서 답변 수집
  • • 결과 분석: 개념별 정답률 계산 및 성취도 분석
  • • 시각화: 지식맵을 통한 학습 현황 시각적 표현

성취도 계산 알고리즘

  • • 개념별 정답률: (정답 수 / 전체 문제 수) × 100
  • • 난이도 가중치: difficulty 값을 활용한 가중 평균
  • • 지식맵 시각화: 정답률에 따른 4단계 색상 구분
  • • Let's Try!(0-25%), Good(26-50%), Great(51-75%), Excellent(76-100%)

역할 및 기여

주요 개발 영역

  • • 데이터 모델링: 문제 데이터 구조 및 평가 로직 설계
  • • 프론트엔드 아키텍처: React + TypeScript 기반 설계
  • • 상태 관리 시스템: Context API 활용 전역 상태 관리
  • • UI/UX 개발: Material-UI와 Styled Components 활용
  • • 문제 유형별 컴포넌트: 4가지 문제 유형 인터랙티브 개발

기술적 도전과 해결

  • • 복잡한 상태 관리: Context API와 useReducer 조합
  • • 동적 지식맵 렌더링: SVG와 CSS Transform 활용
  • • 다양한 입력 방식: 각 문제 유형별 최적화된 UI
  • • 성능 최적화: React.memo와 useCallback 활용
