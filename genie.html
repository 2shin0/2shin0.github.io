지니 - AI 수능 국어 지문/문항 자동 생성 서비스

교육 현장의 문제 출제 업무 경감을 위해 개발된 AI 수능 국어 지문 및 문항 자동 생성 플랫폼입니다.

GitHub 보기 2025.02 ~ 07

프로젝트 개요

지니는 Google Gemini AI를 활용하여 다양한 유형의 지문(단일 지문, 독서론, 복합 지문)과 그에 맞는 문항을 자동으로 생성하며, 교육자들이 효율적으로 평가 자료를 제작할 수 있도록 지원합니다.

팀원: 손명근(기획), 이신영, 이효준, 권홍준, 정도환, 김현경, 최광훈
역할: 기획 및 API 개발

주요 특징

  • Google Gemini 2.5 Pro/Flash를 활용한 지문 생성
  • 단일/독서론/복합 지문 다양한 유형 지원
  • 지문 분석 기반 적합한 문항 자동 생성
  • TipTap 에디터를 통한 실시간 콘텐츠 편집

기술 스택

Frontend

  • • Vue 3 + Composition API
  • • Pinia (상태 관리)
  • • Tailwind CSS, TW Elements
  • • TipTap (텍스트 에디터)
  • • Vue Router 4, Vite

Backend

  • • Spring Boot 3.4.3
  • • Spring Security, OAuth2
  • • MariaDB, JPA/Hibernate
  • • Redis (캐싱)
  • • JWT 인증

AI Server

  • • FastAPI (Python)
  • • Google Gemini 2.5 Pro/Flash
  • • LangChain, OpenAI
  • • FAISS (벡터 검색)
  • • AWS S3 (boto3)

주요 기능

지문 생성

  • • 단일 지문 생성: 특정 주제나 키워드 기반
  • • 독서론 지문 생성: 독서 이론 및 방법론 관련
  • • 복합 지문 생성: 여러 관점이나 자료 종합
  • • 맞춤형 난이도 조절: 학습자 수준별 설정

문항 생성

  • • 지문 분석 기반 문항 생성
  • • 객관식, 주관식, 서술형 등 다양한 유형
  • • AI 기반 문항 적절성 및 난이도 검증
  • • 문항 유형별 참고 예시 제공

사용자 관리

  • • OAuth2 소셜 로그인
  • • JWT 기반 안전한 인증
  • • API 호출 비용 및 사용량 모니터링
  • • 사용자별 기능 접근 권한 제어

콘텐츠 관리

  • • 생성된 콘텐츠 데이터베이스 저장
  • • TipTap 에디터를 통한 실시간 편집
  • • PDF, Word 등 다양한 형식 내보내기
  • • 재사용 가능한 템플릿 관리

주요 컴포넌트

  • • AI 서비스 계층: Gemini API를 활용한 지문/문항 생성
  • • 비즈니스 로직 계층: Spring Boot 기반 핵심 처리
  • • 프레젠테이션 계층: Vue.js 기반 사용자 인터페이스

데이터베이스 스키마

  • • User: 사용자 정보 및 인증 데이터
  • • Passage: 생성된 지문 정보 및 메타데이터
  • • Question: 문항 정보 및 정답 데이터
  • • Template: 재사용 가능한 템플릿 정보
  • • Usage: API 사용량 및 비용 추적 데이터
