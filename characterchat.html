캐미 - AI 기반 인터랙티브 스토리텔링 캐릭터챗봇

사용자는 자신의 캐릭터를 생성하고, AI 기반의 다양한 페르소나와 상호작용하며 스토리텔링 형식으로 퀘스트를 해결합니다.

GitHub 보기 2025.02.11 ~ 2025.05.02

프로젝트 개요

캐미는 사내 AI 에이전트 스터디로 시작되어 진행된 프로젝트입니다. OpenAI GPT 모델과 DALL-E를 활용하여 개인화된 캐릭터 이미지 생성, 동적 스토리 전개, 그리고 몰입감 있는 대화 경험을 제공합니다.

팀원: 이신영, 김현경, 손명근, 정도환
역할: 기획 및 개발

주요 특징

  • DALL-E 3를 활용한 AI 기반 캐릭터 이미지 자동 생성
  • 테마별 스토리 배경 및 동적 세계관 구축
  • 단계별 퀘스트 진행 관리 시스템
  • 다중 AI 캐릭터와의 자연스러운 대화

기술 스택

Frontend

  • • React 19 + TypeScript
  • • Material-UI (MUI)
  • • Zustand (상태 관리)
  • • Emotion, Tailwind CSS
  • • Vite (빌드 도구)

Backend

  • • FastAPI (Python)
  • • OpenAI GPT-4, DALL-E 3
  • • LangChain
  • • PostgreSQL (AsyncPG)
  • • Redis, SQLAlchemy

DevOps

  • • Docker, Docker Compose
  • • Gunicorn, Uvicorn
  • • ESLint, TypeScript
  • • Git 버전 관리

주요 기능

캐릭터 생성 및 커스터마이징

  • • 사용자 맞춤형 캐릭터 생성 (외모, 나이, 직업, 성격)
  • • DALL-E 3를 활용한 AI 기반 캐릭터 이미지 자동 생성
  • • 개성 있는 말투와 성격 설정

동적 세계관 구축

  • • 테마별 스토리 배경 생성
  • • 장르별 세계관 설정 (판타지, SF, 현대 등)
  • • 사용자 캐릭터에 맞춤화된 스토리 환경

퀘스트 시스템

  • • 단계별 퀘스트 진행 관리
  • • 퀘스트 완료 조건 및 성공 기준 설정
  • • 진행 상황 실시간 추적

AI 페르소나 대화

  • • 다중 AI 캐릭터와의 자연스러운 대화
  • • 컨텍스트 기반 응답 생성
  • • 캐릭터별 고유한 성격과 말투 구현
